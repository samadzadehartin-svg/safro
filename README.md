# Safro UI/UX v9 — Photo Edition

نسخهٔ استاتیک و آمادهٔ GitHub Pages.

نسخه ۹ شامل عکس واقعی مقصدها، گالری الهام سفر، کارت‌های تصویری تور، صفحه جزئیات کامل‌تر و کش تازه است.

## انتشار روی GitHub Pages

1. تمام فایل‌های این پوشه را در ریشهٔ مخزن GitHub قرار دهید.
2. در GitHub وارد **Settings → Pages** شوید.
3. در بخش **Build and deployment** گزینهٔ **Deploy from a branch** را انتخاب کنید.
4. شاخهٔ `main` و پوشهٔ `/ (root)` را انتخاب و ذخیره کنید.
5. پس از تکمیل انتشار، لینک GitHub Pages نمایش داده می‌شود.

فایل `index.html` بازدیدکننده را به پنل مشتری هدایت می‌کند. مسیرهای پنل فروش و مدیریت نیز به‌ترتیب `staff/` و `admin/` هستند.
