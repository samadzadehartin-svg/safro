# سفر رو - ساختار فقط سه پنل

در این نسخه فقط سه بخش اصلی جدا شده‌اند:

- `buyer/index.html` پنل خریدار
- `staff/index.html` پنل کارمند
- `admin/index.html` پنل مدیریت

داخل پنل خریدار، صفحه اصلی، جزئیات تور، رزرو، علاقه‌مندی‌ها و رزروهای من همگی در همان یک پنل نمایش داده می‌شوند و فایل جدا ندارند.

رمز دمو:
- کارمند: `staff123`
- مدیریت: `admin123`

برای اجرا بهتر است پوشه را در VS Code با Live Server باز کنید و وارد `buyer/index.html` شوید.


## Vercel

در ریشه پروژه فایل `index.html` و `vercel.json` اضافه شده تا آدرس اصلی سایت به `/buyer/` منتقل شود.


## اصلاح جدید
لینک پنل کارمند و پنل مدیریت از پنل خریدار حذف شد. پنل‌ها فقط با آدرس مستقیم `/staff/` و `/admin/` باز می‌شوند.


## اصلاح جدید
- تقویم حرکت حذف شد و فقط تاریخ‌های حرکت نمایش داده می‌شود.
- ستاره‌های هتل طلایی و زیباتر شدند.
- عکس‌ها به فایل‌های محلی SVG تبدیل شدند تا بدون فیلترشکن هم بالا بیایند.
- تعداد تورها بیشتر شد و تصاویر بر اساس جاهای معروف ساخته شدند.


## اصلاح جدید
در پنل کارمند برای هر بخش گزینه نمایش/عدم نمایش اضافه شد. کارمند می‌تواند بخش‌های توضیحات، اطلاعات پرواز، تاریخ‌ها، هتل‌ها، گالری، برنامه سفر، مدارک، خدمات، قوانین، شرایط کودک و نظر مسافران را فعال یا غیرفعال کند.


## اصلاح جدید
- بالای سایت بخش «مشاوره رایگان / تور خودتو بساز» اضافه شد.
- شماره‌ها در localStorage ذخیره و در پنل مدیریت در بخش «لیست تماس» نمایش داده می‌شوند.
- بعد از مشاوره رایگان، بخش اعتمادسازی اضافه شد.
- فهرست تورها پایین‌تر آمده و چیدمان صفحه خریدار یوزرفرندلی‌تر شد.


## اصلاح جدید
- صفحه اول با بخش «تور خودتو بساز» شروع می‌شود.
- مشاوره رایگان به صورت پاپ‌آپ بعد از مشاهده دو تور نمایش داده می‌شود.
- بعد از ثبت شماره، اطلاعات در لیست تماس مدیریت ذخیره و پیام آماده واتس‌اپ برای شماره 09126144939 باز می‌شود.
- در پنل مدیریت دکمه ارسال کل لیست تماس‌ها به واتس‌اپ مدیر اضافه شد.
- ارسال پیامک خودکار واقعی در نسخه استاتیک Vercel ممکن نیست و نیاز به بک‌اند و پنل پیامکی دارد.


## اصلاح جدید
