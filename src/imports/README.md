# سفرو - ساختار فقط سه پنل

در این نسخه فقط سه بخش اصلی جدا شده‌اند:

- `buyer/index.html` پنل مشتری
- `staff/index.html` پنل فروش
- `admin/index.html` پنل مدیریت

داخل پنل مشتری، صفحه اصلی، جزئیات تور، رزرو، علاقه‌مندی‌ها و رزروهای من همگی در همان یک پنل نمایش داده می‌شوند و فایل جدا ندارند.

رمز دمو:

- کارمند: `staff123`
- مدیریت: `admin123`

برای اجرا بهتر است پوشه را در VS Code با Live Server باز کنید و وارد `buyer/index.html` شوید.

## Vercel

در ریشه پروژه فایل `index.html` و `vercel.json` اضافه شده تا آدرس اصلی سایت به `/buyer/` منتقل شود.
