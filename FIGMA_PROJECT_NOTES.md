# Figma Safro Project Notes

این فایل خروجی Figma Make به صورت React + Vite است، نه فایل خام `.fig`.

## اصلاحات انجام‌شده

- فایل `src/main.tsx` که خالی بود تکمیل شد.
- فایل `vite.config.ts` که خالی بود تکمیل شد.
- فایل `tsconfig.json` که خالی بود تکمیل شد.
- فایل `src/vite-env.d.ts` تکمیل شد.

## اجرا

```bash
npm install
npm run dev
```

بعد آدرس local را در مرورگر باز کنید.

## مسیرها

- `/` صفحه مشتری
- `/staff` صفحه کارشناس
- `/admin` صفحه مدیریت
