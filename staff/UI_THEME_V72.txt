Safro v7.2 — data-theme Dark Mode Integration

- data-theme="dark|light" روی عنصر html منبع اصلی تم است.
- کلاس‌های body.dark و body.dark-mode برای سازگاری با CSS قدیمی همزمان نگه داشته می‌شوند.
- انتخاب کاربر در localStorage با کلید safro-theme ذخیره می‌شود.
- تنظیم قدیمی safarro_three_panels_dark به‌صورت خودکار مهاجرت می‌شود.
- اسکریپت کوچک قبل از CSS از چشمک روشن هنگام بارگذاری دارک‌مود جلوگیری می‌کند.
- آیکون ماه/خورشید و aria-label به‌صورت خودکار هماهنگ می‌شوند.
- انیمیشن کارت‌ها، هدر شیشه‌ای و منوی موبایل نسخه قبلی حفظ شده‌اند.
