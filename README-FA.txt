نسخه مستقل Safro UI/UX v9.0 — نسخه کامل تصویری

روش اجرا:
1) فایل ZIP را کامل Extract کنید.
2) پوشه استخراج‌شده را با Live Server یا هر وب‌سرور ساده اجرا کنید.
3) index.html را باز کنید؛ به‌صورت خودکار وارد پنل مشتری می‌شود.

مسیرها:
- پنل مشتری: buyer/index.html
- پنل فروش: staff/index.html
- پنل مدیریت: admin/index.html

این نسخه دارای عکس واقعی مقصدها، گالری تصویری، کارت‌های جدید و طراحی واکنش‌گرا است.
اگر ظاهر قدیمی دیده شد، Ctrl + Shift + R را بزنید. نسخه کش این بسته v9.0 است.
