# راهنمای اتصال Supabase

1. در Supabase یک پروژه بساز.
2. فایل `supabase_schema.sql` را در SQL Editor اجرا کن.
3. فایل `assets/js/config.js` را باز کن و `SAFARO_SUPABASE_URL` و `SAFARO_SUPABASE_ANON_KEY` را جایگزین کن.
4. سایت را Deploy کن.
5. وارد `/admin/` شو و روی «ارسال همه» بزن.

رمزها:
مدیریت: `admin123 / admin123`
کارشناس: `staff123 / staff123`
